# Imitation Login Page

### In Terminal console:

```bash
  git clone https://github.com/saburchik/only.git
```

```bash
  npm install
```

- Plus, install [json.server](https://github.com/typicode/json-server)

```bash
  npm install -g json-server
```

- To Start:

```bash
  npm start
```

```bash
  json-server --watch db.json --port 3004
```
